## Development
### Running Application in Development
```bash
.\gradlew bootRun
```

```bash
$ docker build -t neorekruit .
```
### Running the Container
```bash
$ docker run -e POSTGRES_JDBC="<your_value>" \
-e POSTGRES_PASSWORD="<your_value>" \
-e POSTGRES_USERNAME="<your_value>" \
-p 8080:8080 neorekruit
```
